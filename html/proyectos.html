Proyectos

El molcajete es una página web dinámica que ofrece una variedad de platillos deliciosos y nutritivos para todos los gustos y ocasiones. Los usuarios pueden registrarse o iniciar sesión en la página web para acceder a las diferentes opciones y servicios que ofrece. Pueden explorar el catálogo de platillos, que cuenta con animaciones, imágenes, precios y descripciones de cada uno. También pueden ver las especialidades de la casa en un slider que muestra los platillos más populares y recomendados. Los usuarios pueden agregar los platillos que deseen al carrito de compras, donde pueden ver el detalle y el total de su pedido. Una vez que estén listos para comprar, pueden confirmar su pedido y recibir una notificación de confirmación. Los usuarios también pueden consultar el historial de sus pedidos en el apartado de pedidos, donde pueden ver el estado y el seguimiento de cada uno. Además, la página web cuenta con un lado de administrador, donde se puede gestionar el inventario de platillos, agregar platillos nuevos, ver el registro de usuarios y las compras realizadas, y generar reportes estadísticos.

Mi Grúa de Arduino es un proyecto que consiste en construir y programar una grúa con materiales reciclados y un Arduino. La grúa tiene una pinza que puede abrirse y cerrarse para agarrar objetos, y un brazo que puede moverse en cuatro direcciones : izquierda, derecha, arriba y abajo. La grúa se controla con un joystick HP y dos botones, que se conectan al Arduino mediante cables. El joystick tiene dos ejes: el eje X y el eje Y. El eje X se usa para mover el brazo de la grúa hacia la izquierda o la derecha, y el eje Y se usa para mover el brazo hacia arriba o abajo. Los dos botones se usan para abrir y cerrar la pinza de la grúa. El proyecto se realizó con Arduino, un joystick HP, dos botones, cables, palitos de madera y papel cascarón.

Sistema de Inventario es un proyecto que consiste en desarrollar y programar una aplicación web y móvil que permite gestionar el inventario de productos, los usuarios y las compras. La aplicación usa Cordova para crear una interfaz multiplataforma que se adapta a diferentes dispositivos y navegadores. La aplicación usa Web SQL para almacenar y manipular los datos de los productos, los usuarios y las compras. La aplicación permite ingresar productos, cantidades y precios a una tabla, donde se puede ver el detalle y el total del inventario. La aplicación también permite crear usuarios y asignarles una cantidad de crédito, que se les descuenta en base al precio del producto que compran. La aplicación también permite ver el historial de compras de cada usuario y el saldo de su crédito.

Hacienda Belmont es un videojuego de aventura y acción, que se desarrolla en un mundo fantástico y misterioso. El jugador asume el rol de Alex Turner, un joven valiente y curioso, que se ve envuelto en una trama de intrigas y peligros, cuando descubre que su padre es en realidad Belmont, el villano más temido y poderoso del reino. Alex tendrá que enfrentarse a samuel Belmont, explorar diferentes escenarios,recolectar objetos para descubrir los secretos de su origen y El videojuego cuenta con una historia envolvente, unos personajes carismáticos, una música ambiental, un menú de opciones y un gameplay con mecánicas variadas, interacción con el entorno y objetivos claros.

Mi Sistema de Control de Activos es un sistema que permite registrar, asignar, modificar y eliminar los activos de una organización, así como los usuarios que los utilizan. El sistema se conecta a una base de datos PHPMyAdmin, donde se almacenan y consultan los datos de los activos, los usuarios y las asignaciones. El sistema tiene una interfaz gráfica con un bonito diseño, que facilita el uso y la navegación. El sistema permite realizar las siguientes operaciones Registrar usuarios: se puede ingresar el nombre, el correo, el rol y la contraseña de cada usuario, y se genera un identificador único para cada uno. Registrar productos: se puede ingresar el nombre, la descripción, el tipo, el estado y el valor de cada producto, y se genera un código de barras para cada uno.
